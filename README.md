# Tvheadend Movistar+ Astra 19.2E config script

## Usage

### Backup
tvheadend -b

### Install
tvheadend -i

### Save
tvheadend -s

## Picons
Picons shall be obtained from [OpenLD/Astra19.2-Full-Transparent](https://github.com/OpenLD/Astra19.2-Full-Transparent)
